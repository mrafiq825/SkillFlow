## [1.0.0] - 2026-07-12

### Added
- First public release of `skills-cli`.
- Implement `init` command to scaffold `.agents` configurations.
- Implement `create` command to create skills directories and files.
- Implement `list` command to render a terminal table of active customizations.
- Implement `validate` command to enforce YAML frontmatter validation and body length limits.
- CI/CD, security scanning, dependabot, and bug report templates.
- Native Node.js test runner integration test suite.
- Added `combined-security-audit` skill and integrated task-completion rules.
