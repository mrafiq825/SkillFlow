# Security Policy

This document describes supported versions of `skills-cli` and how to report vulnerabilities.

## Supported Versions

Only the latest major version receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do **NOT** report it publicly through issues.

Instead, please send an email with details of the vulnerability to:
**security@example.com**

Please include:
- Steps to reproduce or proof-of-concept code.
- Impact description.
- Affected environment and version.

We will acknowledge receipt within 48 hours and coordinate a public release timeline with responsible disclosure.
