# Support Policy

This document outlines how to get assistance, ask questions, or report problems with `skills-cli`.

## Ask a Question
If you have questions about how to use `skills-cli`, please use:
- **GitHub Discussions**: Visit our repository discussion board to ask questions and share feedback with the community.
- **FAQ Section**: Check the FAQ section in the [README](file:///Users/muhammadrafiq/Desktop/Agent%20Skills/skills-cli/README.md) to see if your question is already answered.

## Bug Reporting & Feature Requests
- **Bugs**: If you encounter unexpected behavior or errors, please check existing issues before opening a new one. File bug reports using the [Bug Report Template](https://github.com/muhammadrafiq/skills-cli/issues/new?template=bug_report.yml).
- **Features**: Propose enhancements or new ideas using the [Feature Request Template](https://github.com/muhammadrafiq/skills-cli/issues/new?template=feature_request.yml).

## GitHub Repository Security & Protection Settings
For maintainers of this repository, it is recommended to configure:
- **Default Branch**: Set `main` as the default branch.
- **Branch Protection**: Enforce branch protection rules on `main` to require at least one approving pull request review and successful CI checks before merging.
- **Signed Commits**: Enforce commit signature verification to guarantee code authenticity.
